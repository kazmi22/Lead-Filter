# LeadFilter 🚀

An AI-driven Micro-SaaS tool for Real Estate intelligence. This app analyzes and prioritizes incoming customer inquiries to help agents focus on high-intent leads.

## Features
- **Lead Prioritization:** Automatically filters "human" lead scenarios.
- **Fast Logic:** Lightweight backend for quick response times.
- **Simple UI:** Easy-to-use frontend for real estate teams.

## Tech Stack
- **Backend:** Python
- **Frontend:** HTML/CSS
